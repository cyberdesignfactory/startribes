# Star Tribes

## domain/game

This contains the game logic.  Several discrete unit-tested features reside here, which are brought together in a "cycle".  Each feature (and the overall game cycle) transforms a "world" data structure into a new data structure.

To run tests, the Clojure command line tools must be installed, then use the command:

clj -X:test
